# 📚 Aniket’s Study Saga

**Smart Prep for Smarter Students** ✨  
An AI-powered exam preparation assistant built for engineering students. It analyzes past papers, predicts future questions unit-wise, and helps you revise faster and smarter!

---

## 🚀 Features

- 🤖 **AI-like Question Prediction**  
  Upload 2+ PDFs, and the app will predict questions by analyzing patterns.

- 🧠 **Unit-wise Predictions**  
  Questions are grouped based on Q1, Q2… for realistic practice.

- 📄 **Downloadable PDF**  
  All predicted questions are saved as a neat, printable PDF: `PredictionbyAniket.pdf`.

- 🎨 **Interactive UI with Animations**  
  Lottie animations and themed subject pages for an engaging experience.

- 💬 **Ask Anything**  
  Type any question into the subject page and get instant AI-style answers (mock or GPT-ready).

- 📊 **Analytics Page**  
  View top repeated concepts across all subjects using NLP-based concept extraction.
